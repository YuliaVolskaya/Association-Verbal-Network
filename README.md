# Количественный и структурный анализ ассоциативно-вербальной сети русского языка
Проект реализуется в рамках программы ДПО "Компьютерная лингвистика" (НИУ "Высшая школа экономики") 2025-2026 гг. Материалом исследования служат данные ассоциативного эксперимента, который проводится в рамках проекта по созданию Нового ассоциативного словаря русского языка при поддержке РНФ (проект № 25-78-00016, https://rscf.ru/project/25-78-00016/). Автор проекта: Вольская Юлия Александровна, канд.филол.н., доцент кафедры прикладной и экспериментальной лингвистики, Казанский (Приволжский) федеральный университет, г.Казань. 
## Введение
Ассоциативный эксперимент – один из базовых методов исследования в области психолингвистики, который используется с целью всестороннего изучения особенностей языкового сознания, ментального лексикона, ценностных ориентиров, основополагающих особенностей мировоззрения; для анализа вопросов этнопсихолингвистики; в исследованиях, направленных на осуществление социальных программ; с целью изучения лексических и синтаксических особенностей слов. Как правило, для реализации подобных исследований создаются обширные базы данных ассоциаций или же словари ассоциаций, которые составляются на основе массовых экспериментов.
Основная **цель проекта** – произвести обработку, частотный и сетевой анализ сформированной ассоциативно-вербальной сети (термин Ю. Н. Караулова).
## Корпус и инструменты исследования
Общий корпус: около 248 000 ассоциативных связей. После первичной обработки (объединение повторяющихся ассоциаций) объем корпуса уменьшился до 77 340 тыс. ассоциативных связей. 

**Используемые библиотеки**

| Назначение | Библиотеки |
|-----------|-----------|
| Анализ данных | `pandas`, `numpy` |
| Лингвистическая обработка | `pymystem3`, `nltk` |
| Визуализация | `matplotlib`, `seaborn`, `wordcloud` |
| Сетевой анализ | `networkx` |

**Метрики и алгоритмы**:
* TTR (по формуле Ю.Н. Караулова)
* TF-IDF
* коэффициентом пересечения Диза
* degree, weighted degree, degree centrality, betweenness centrality, closeness centrality
* greedy modularity maximization
* алгоритм Фрухтермана–Рейнгольда
## Ход исследования и основные результаты
### Предварительная обработка и нормализация корпуса 
1. Необработанные данные эксперимента загружались в формате XLSX
2. На начальном этапе массив первичных ассоциативных данных подвергался стандартизации. Процедура включала приведение всех реакций к нижнему регистру, удаление нерелевантных символов, объединение повторяющихся ассоциаций и указание их абсолютных частот.
3. Производилась трансформация формата датафрейма для удобства дальнейшего статистического анализа и визуализации: преобразование исходной таблицы (строки – респонденты, а столбцы – стимулы) в длинный формат: стимулы – в первом столбце, ассоциации – во втором, частоты – в третьем.
4. Получен датафрейм df2.

## Лемматизация и морфологический анализ 
1. Для перехода от анализа словоформ к анализу концептов все ассоциации были лемматизированы и сохранены в отдельную колонку.
2. Для категориального анализа ассоциативной сети ассоциации и стимулы получили частеречную разметку (pymystem3). При частеречной разметке ассоциациям из более чем одного слова приписывались теги через запись в формате ч.р.+ч.р.(напр., V + V).
3. На этом этапе обновлен датафрейм df2, к исходным данным добавлены столбцы с леммой, частью речи леммы и стимула. 

## Анализ на уровне лемм
1. Реализован пересчет частот лемм (объединяются частоты словоформ, относящиеся к данной лемме). Сформирован новый датафрейм df_lemmas с парами стимул-ассоциация и обновленными частотами. Это позволяет увидеть общую картину ассоциативных связей. 
2. Выявлены пересекающиеся леммы, то есть такие, которые встречались в качестве ассоциаций у двух и более разных стимулов. Такие леммы формируют ядро ассоциативно-вербальной сети. Для каждой леммы рассчитывалось количество связанных стимулов. Сформирован новый датафрейм shared_lemmas.

## Квантитативный анализ ассоциативной сети на разных уровнях
1. Расчет следующих показателей для каждого стимула: а) всего ответов, б) количество различных ассоциаций, в) количество единичных ассоциаций; г) количество отказов.
2. Коэффициент лексического разнообразия по формуле Ю.Н. Караулова для каждого стимула (отношение разных ассоциатов к общему количеству ответов).
3. Вычисление среднего TTR по всему списку стимулов и отдельно для стимулов разных частей речи (существительные, глаголы, прилагательные). 
4. Частеречный анализ ассоциаций: для каждого стимула подсчитано процентное соотношение ассоциаций, принадлежащих к разным частям речи. Это позволяет рассмотреть грамматическую структуру ассоциативного поля каждого стимула.
5. На основе этих показателей сформирован новый датафрейм df_summary.
6. Анализ ядерных ассоциаций, то есть выявление самых сильных связей. Для каждого стимула была определена наиболее частотная ассоциация, рассчитана «сила связи» (в процентах) как отношение частоты самой частотной ассоциации к общему числу ответов на данный стимул. Сформирован отдельный датафрейм df_strongest.
7. Использование меры TF-IDF для каждой леммы в рамках ассоциативного поля к каждому стимулу. Это позволяет выявить ассоциации, наиболее специфичные и уникальные для конкретного стимула, в противовес общим, не уникальным, часто встречающимся во всем корпусе. Сформирован отдельный датафрейм df_tfidf.

## Сетевой анализ 
**Материал**: сетевой анализ реализовывался на основе лемматизированных ассоциаций. 
**Основная идея**: два стимула связаны тем сильнее, чем больше общих ассоциаций они вызывают и чем выше частоты этих ассоциаций. 
В данном случае сеть является **неориентированным графом**, использовалась **однодольную проекцию двудольного графа**. Узлы – стимулы, у которых есть совпадающие леммы-ассоциации. Ребра – общие ассоциации. 
Анализ позволяет рассматривать структуру связей между стимулами, находить центральные стимулы и кластеры стимулов на основе общности их ассоциативных полей. Неориентированная сеть сходства только между стимулами служит основной для анализа структуры ассоциативного тезауруса.
**Сила ребер** рассчитывалась по формуле, предложенной **Дж. Дизом** для количественной оценки сходства ассоциативных полей слов.

, где: N(A) – общее количество реакций на стимул A (сумма частот всех ассоциаций), N(B) – то же для стимула B, числитель – сумма минимальных частот по каждой общей реакции: fap — частота, с которой реакция p встречается у стимула A (т.е. сколько раз испытуемые дали ответ p на стимул A), fbp — частота, с которой та же самая реакция p встречается у стимула B.

Использовались базовые сетевые метрики (degree, weighted degree, degree centrality, betweenness centrality, closeness centrality) и сформирован отдельный датафрейм df_graph_metrics.

**Построение графа**
Для визуализации сеть фильтровалась по порогу веса (threshold_k = 0.018), чтобы убрать слабые, возможно, шумовые связи. Затем применялся алгоритм обнаружения сообществ greedy modularity maximization, реализованный в NetworkX. Это ползволило выделить группы узлов, внутри которых связи плотнее, что соответствует семантическим категориям или тематическим кластерам.

Для компоновки графа используется силовой алгоритм Фрухтермана–Рейнгольда (Fruchterman–Reingold, 1991) – spring_layout с параметром k=3.0. Алгоритм моделирует систему, в которой узлы отталкиваются друг от друга, а рёбра притягивают, что позволяет визуально разделить слабосвязанные компоненты и подчеркнуть структуру групп. Чтобы дополнительно выделить кластеры, узлы каждого сообщества сначала укладываются локально, а затем смещаются в разные стороны (group_centers), что делает группировку более наглядной.
